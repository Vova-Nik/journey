To obtain token use:
{
"email": "admin@ttt.ua",
"password": "123"
}
Possible get params
  • sortColumn = "id", "name", ... entity specific
  • sortDirection = "asc", "desc"
  • filterField = "id", "name", ... entity specific
  • filterValue = "*", ... tntity specific;
  • filterOperation = GREATER, LESS, NOT_EQUALS, EQUALS, LIKE
  • pageNumber = decimal
  • pageSize = decimal (DEFAULT_PAGE_SIZE =8)
Внимание!. В рамках одной сущности не заданные в запросе параметры могут оставаться от предыдущих хапросов.
Нумерация страниц в get с нуля, в view с единицы.
